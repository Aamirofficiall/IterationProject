# IterationProject

## How to install

1- Clone git repository "git@github.com:deyi83/IterationProject.git"
<br>
2- Run "pip install -r requirements.txt"
<br>
3- Run "python manage.py migrate"
<br>
4- Run "python manage.py runserver"
